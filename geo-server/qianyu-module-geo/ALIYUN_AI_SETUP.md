# 蒸馏词AI模型配置说明

## 默认配置：豆包模型

系统默认使用**豆包模型**进行蒸馏词生成，配置如下：

```yaml
ai:
  llm:
    doubao-api-key: bc16a9bb-397b-424b-8d03-b613fe8a71ff  # 豆包API Key
    use-aliyun: false  # 使用豆包模型
```

## 可选配置：阿里云通义千问

如果需要切换到阿里云通义千问，按以下步骤配置：

### 1. 获取阿里云API Key

1. 访问 [阿里云百炼平台](https://bailian.console.aliyun.com/)
2. 开通通义千问服务
3. 创建API Key，复制保存

### 2. 修改配置

在 `application-local.yaml` 中修改：

```yaml
ai:
  llm:
    use-aliyun: true  # 切换到阿里云
    aliyun-api-key: sk-xxxxxxxxxxxxxxxxxxxxx  # 你的阿里云API Key
    aliyun-model: qwen-plus  # 可选: qwen-turbo、qwen-plus、qwen-max
```

### 3. 可用模型

- `qwen-turbo`: 速度最快，成本最低
- `qwen-plus`: 性价比最高（推荐）
- `qwen-max`: 性能最强，成本最高

## 智能降级机制

系统具有多层降级机制，确保服务稳定性：

1. **首选**: 配置的AI模型（豆包或阿里云）
2. **降级1**: 如果配置了`use-aliyun: true`但阿里云失败，降级到豆包
3. **降级2**: 如果所有AI都失败，降级到模板方式
4. **最终**: 模板方式作为兜底方案
